PDFBrute — PDF Password Recovery

What is PDFBrute? PDFBrute is a highly secure, client-side tool to recover access to your encrypted PDFs on-device using custom password patterns. It uses multi-core Web Workers and a bidirectional search algorithm for blazing fast performance. 100% private, no install required, and your file never leaves your browser.

Key Features

  • 100% Privacy: Files are processed locally on your device.
  • Multi-Core Parallelization: Uses all available CPU cores via Web Workers.
  • Bidirectional Search: Cuts worst-case search times in half by squeezing the search space.
  • No Install: Pure web application accessible from any modern browser.
